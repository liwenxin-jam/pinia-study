# visual-demo

- 可视化界面，接口数据来源于根目录下的node-demo。
- 摸鱼神器
  1. json2ts (快捷键 ctrl+alt+v)
  2. json to ts (快捷键 ctrl+shift+alt+s)

## 构建项目
```sh
npm init vue@latest
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
