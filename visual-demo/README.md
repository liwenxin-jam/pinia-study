# visual-demo

- 可视化界面，接口数据来源于根目录下的node-demo。其中用了摸鱼神器json2ts(快捷键 ctrl+alt+v)

## 构建项目
```sh
npm init vue@latest
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
